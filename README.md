# MWA3EZ - Reels Backend API

## وصف المشروع

MWA3EZ هو API خلفي لتطبيق Reels (فيديوهات قصيرة) مشابه لـ Instagram Reels أو TikTok. يدير الفيديوهات اليومية مع إحصائيات المشاهدات والإعجابات، ويوفر إحصائيات إدارية.

## الميزات

- إدارة الفيديوهات (إضافة، حذف، عرض)
- تتبع المشاهدات والإعجابات
- تحديث يومي تلقائي للفيديوهات
- إدارة المستخدمين عبر ملفات تعريف الارتباط
- توثيق API باستخدام Swagger
- إحصائيات إدارية

## متطلبات التشغيل

- Node.js (إصدار 14 أو أحدث)
- npm أو yarn

## التثبيت

1. قم بتثبيت التبعيات:
```bash
npm install
```

2. شغل الخادم:
```bash
npm start
```

للتطوير مع إعادة التحميل التلقائي:
```bash
npm run dev
```

## الاستخدام

الخادم يعمل على `http://localhost:4000`

### نقاط النهاية الرئيسية

#### الفيديوهات
- `GET /api/videos` - الحصول على جميع الفيديوهات
- `POST /api/videos` - إضافة فيديو جديد
- `GET /api/videos/:id` - الحصول على فيديو محدد
- `DELETE /api/videos/:id` - حذف فيديو
- `GET /api/videos/last` - الحصول على آخر فيديو تم مشاهدته
- `POST /api/videos/:id/views` - زيادة عدد المشاهدات
- `POST /api/videos/:id/likes` - زيادة عدد الإعجابات

#### الإدارة
- `GET /api/admin/stats` - إحصائيات الإدارة
- `POST /api/admin/refresh` - تحديث الفيديوهات يدوياً
- `GET /api/admin/videos` - عرض جميع الفيديوهات (إداري)

### توثيق API

يمكن الوصول إلى توثيق API الكامل عبر Swagger على:
`http://localhost:4000/api/docs`

## هيكل المشروع

```
backend/
├── app.js          # نقطة دخول التطبيق
├── routes.js       # تعريف نقاط النهاية
├── videoStore.js   # إدارة بيانات الفيديوهات
├── package.json    # تبعيات المشروع
└── data/
    └── videos.json # تخزين بيانات الفيديوهات
```

## التبعيات

### التبعيات الأساسية
- `express` - إطار العمل الرئيسي
- `cors` - دعم CORS
- `cookie-parser` - معالجة ملفات تعريف الارتباط
- `swagger-jsdoc` & `swagger-ui-express` - توثيق API

### تبعيات التطوير
- `nodemon` - إعادة التحميل التلقائي أثناء التطوير

## التحديث اليومي

يقوم النظام بتحديث الفيديوهات تلقائياً كل يوم عند منتصف الليل، حيث يقوم بإنشاء 10-20 فيديو عشوائي جديد.
